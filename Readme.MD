# ODK Solutions Hub — Official Website

**Obeng Dominic Koranteng Solutions Hub**

A modern, responsive website for ODK Solutions Hub — providing expert data analysis training, mathematics tutoring, academic research support, and international test preparation.

## 🚀 Getting Started

### Upload to GitHub Pages

1. Create a new repository on GitHub (e.g. `odksolutionshub`)
2. Upload both files:
   - `index.html`
   - `odk-logo.png`
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch → `/ (root)` → Save
5. Your site will be live at: `https://yourusername.github.io/odksolutionshub`
